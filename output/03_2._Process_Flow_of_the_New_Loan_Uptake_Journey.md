# 2. Process Flow of the New Loan Uptake Journey

```mermaid
graph LR
    A[User Logs into App] --> B{Personalized Loan Offer Displayed?};
    B -- Yes --> C[View Loan Offer];
    B -- No --> D[Browse Loan Products];
    C --> E[Eligibility Check];
    E -- Eligible --> F[Apply for Loan];
    E -- Not Eligible --> G[Reason for Ineligibility];
    D --> E;
    F --> H[Loan Application Submitted];
    H --> I[Application Processing];
    I --> J[Approval/Rejection Notification];
    G --> K[Further Information/Alternative Products];
    K --> D;
```
